### Dataset
Download [CardiacArrhythmia dataset](https://physionet.org/content/challenge-2017/1.0.0/training2017.zip) and unzip it into `data/CardiacArrhythmia/` folder.


### Train models
```bash
make train
```

### Generate verification problems
```bash
make spec
```

### Verify problems
```bash
make verify
```
